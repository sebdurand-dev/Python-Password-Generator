# Simple Python Password Generator

## ✨ Features
*   **User Input:** Allows the user to set their desired password length.
*   **Varied Generation:** Generates the password with a random combination of symbols, lettters, and numbers.
*   **Error Handling:** Catches invalid inputs.
  
## 🚀 How to Use
1.  **Copy files:** Copy the repositories scripts and text file.
2.  **Setup:** Paste/import them into your favorite code editor.
3.  **Run:** Execute the main.py file and enjoy!

## 🛠 Technologies Used
*   **Python 3**
